## Installation

1. Clone the repository
2. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Ensure EZ-Robot ARC software is installed and configured
4. Run the application:
   ```bash
   python main.py
   ```

## Usage

### Basic Operation
1. Start CARL using the "Run Bot" button
2. Speak to CARL through the microphone
3. CARL will respond with both speech and physical actions

## Ethical Considerations

CARL is designed as an augmentative tool for education, research, and personal robotics, not for deception or manipulation. All data is stored locally and not shared externally. The system is intended to be supportive, informative, and respectful of user autonomy.
